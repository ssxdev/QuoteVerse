<p>1. Clone the repository</p>

```

```


<p>3. Install dependencies</p>

```

```

<p>4. Start the NodeJs development server</p>


<p>6. Install dependencies</p>

<p>7. Start the ReactJs development server</p>


<p>8. Open your web browser and visit</p>

```
http://localhost:3000
```
